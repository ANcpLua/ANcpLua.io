# AL0023: Unsupported form type

The type marked with `[FromForm]` cannot be form-bound.

## When it triggers

```csharp
// Error: Parameter 'data' on 'Submit' cannot be form-bound: interfaces cannot be form-bound
public class Controller
{
    void Submit([FromForm] IMyData data) { }  // AL0023
}

// Error: abstract types cannot be form-bound
public abstract class AbstractData { }
public class Controller
{
    void Submit([FromForm] AbstractData data) { }  // AL0023
}
```

## Why this matters

Form binding requires the ability to:
1. Create an instance of the type
2. Set properties from form field values

Types that cannot be instantiated or lack appropriate constructors fail at runtime.

## Bindable types

The following types can be used with `[FromForm]`:
- Primitive types (`string`, `int`, `bool`, `Guid`, etc.)
- Nullable primitives (`int?`, `DateTime?`, etc.)
- Classes with a parameterless constructor
- Classes with a primary constructor where all parameters are primitives
- `IFormFile` and `IFormFileCollection` (special handling)

## How to fix

Use a concrete class with a suitable constructor:

```csharp
// Option 1: Parameterless constructor
public class FormData
{
    public string Name { get; set; }
    public int Count { get; set; }
}

// Option 2: Primary constructor with primitives
public record FormData(string Name, int Count);

public class Controller
{
    void Submit([FromForm] FormData data) { }
}
```

## Configuration

```ini
[*.cs]
dotnet_diagnostic.AL0023.severity = error
```
