# AL0010 - Type should be partial

## Description

Types that are targets for source generators should be marked as partial.

## Properties

- **Category**: Design
- **Severity**: Info
- **Enabled by default**: False
- **Code fix available**: True

## Configuration

```editorconfig
dotnet_diagnostic.AL0010.severity = warning
```
