# AL0020: IFormCollection requires explicit [FromForm]

`IFormCollection` parameters must have an explicit `[FromForm]` attribute for form binding to work correctly.

## When it triggers

```csharp
// Error: Parameter 'form' on 'Submit' uses IFormCollection but lacks [FromForm]
public class Controller
{
    void Submit(IFormCollection form) { }  // AL0020
}
```

## Why this matters

Unlike `IFormFile` which auto-binds in ASP.NET Core Minimal APIs, `IFormCollection` requires explicit attribution:
- Explicit intent makes the binding source clear
- Prevents confusion with other collection types
- Aligns with the principle of explicit parameter binding

## How to fix

Add the `[FromForm]` attribute:

```csharp
public class Controller
{
    void Submit([FromForm] IFormCollection form) { }
}
```

## Configuration

```ini
[*.cs]
dotnet_diagnostic.AL0020.severity = error
```
