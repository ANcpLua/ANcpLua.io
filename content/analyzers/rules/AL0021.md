# AL0021: Multiple structured form sources

An endpoint can only have one structured form bodyâ€”either a `[FromForm]` DTO or `[FromForm] IFormCollection`.

## When it triggers

```csharp
// Error: Endpoint 'Submit' has multiple [FromForm] parameters
public class Controller
{
    void Submit([FromForm] FormData data1, [FromForm] FormData data2) { }  // AL0021
}
```

## Why this matters

HTTP requests have a single request body. Having multiple `[FromForm]` DTOs creates ambiguity:
- Which parameter receives which form fields?
- How should overlapping field names be handled?
- The binding behavior becomes unpredictable

## How to fix

Combine the data into a single DTO:

```csharp
public class CombinedFormData
{
    public string Field1 { get; set; }
    public string Field2 { get; set; }
}

public class Controller
{
    void Submit([FromForm] CombinedFormData data) { }
}
```

Or use `[AsParameters]` for complex scenarios:

```csharp
public record FormRequest(
    [FromForm] string Name,
    [FromForm] int Count,
    IFormFile Attachment);

public class Controller
{
    void Submit([AsParameters] FormRequest request) { }
}
```
