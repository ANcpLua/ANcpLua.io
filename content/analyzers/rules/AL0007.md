# AL0007 - GetSchema should be explicitly implemented

## Description

Implement IXmlSerializable.GetSchema explicitly to avoid exposing it as part of the public API.

## Properties

- **Category**: Usage
- **Severity**: Error
- **Enabled by default**: True
- **Code fix available**: False

## Configuration
